Class: Place

app.Place

new Place(lat, lng, name, suburb, address, image, description, open_now, phone, fourSquareUrl, yelpSnippet, yelpUrl)

Represents the results from one or more APIs. Essentially just a vanilla object with its properties specified as parameters for more explicit code.
Parameters:
Name Type Description
lat number The latitude of the place.
lng number The longitude of the place.
name string The name of the place.
suburb string The suburb / city of the place.
address string The address of the place.
image string This URL of an image of the place.
description string The type of place this is eg. Bike Shop.
open_now boolean Open / closed status of the place at time of API call.
phone string The phone number of the place
fourSquareUrl string URL of Foursquare page for this place.
yelpSnippet string A review snippent from the Yelp API results.
yelpUrl string A URL of the Yelp page for this place.
